# 情侣同居日常：自然居所专题

## 场景概述

本设计基于自然环境参数，在亚热带湿润森林的河畔云居上，设计一个与自然融合的二人世界居所。详细设计居所的各个房间和功能区，通过在不同房间中与AI助手对话来体验和练习相关英语表达。所有场景名称使用中文，贴近真实生活。

## 环境参数

| 项目 | 值 |
| --- | --- |
| 坐标 | (174, 254) |
| 地形 | 陆地、平原 |
| 海拔 | 高海拔 (相对高度 +0.726) |
| 气候带 | 亚热带 |
| 湿度 | 湿润 |
