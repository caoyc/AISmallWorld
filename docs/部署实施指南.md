# 部署实施指南：数字家园试炼场

**按顺序执行以下步骤，每一步都必须完成后再进入下一步。所有命令直接复制粘贴执行即可。**

## 步骤 1：克隆项目

```bash
git clone https://github.com/caoyc/AISmallWorld.git   # 克隆仓库
cd AISmallWorld                                        # 进入项目目录
```

## 步骤 2：创建 conda 环境并安装依赖

```bash
conda create -n aismallworld python=3.12 -y   # 创建 Python 3.12 环境（Evennia 要求 3.12+）
conda activate aismallworld                    # 激活环境
pip install evennia websockets beautifulsoup4  # 安装游戏引擎和依赖
export PYTHONIOENCODING=utf-8                  # 终端输出使用 UTF-8，避免中文乱码
```

> `conda activate` 不可用时，用全路径替代 `python` 和 `evennia`：
> ```bash
> conda info --envs    # 查看 aismallworld 环境的安装路径
> ```
> 找到 `aismallworld` 那行的路径，`python` 替换为 `<路径>/python.exe`，`evennia` 替换为 `<路径>/Scripts/evennia.exe`。

## 步骤 3：初始化数据库

```bash
cd AISmallWorld/evennia_game   # 进入 Evennia 实例目录
evennia stop                    # 确保服务未运行（未运行时不会报错，安全操作）
mkdir -p server/logs            # 创建日志目录（clone 后不存在，不创建会报错）
rm -f server/evennia.db3        # 删除残留数据库（首次部署时文件不存在也无所谓，-f 不会报错）
evennia migrate                 # 初始化数据库表结构
```

看到所有迁移都输出 `OK` 就继续。

## 步骤 4：启动 Evennia 并创建管理员

```bash
cd AISmallWorld/evennia_game
export EVENNIA_SUPERUSER_USERNAME=mofi                     # 管理员用户名
export EVENNIA_SUPERUSER_PASSWORD='Mofi#2026Island'        # 管理员密码（已验证通过 Django 密码校验）
export EVENNIA_SUPERUSER_EMAIL=                             # 邮箱留空
evennia start                                                # 启动服务，首次启动自动创建管理员
```

等待输出 `Server started.` 和 `Portal started.`。

确认三个端口就绪：

```bash
netstat -an | grep -E ":400[0-2].*LISTENING"   # 检查端口 4000/4001/4002
```

必须看到 4000、4001、4002 三个端口都在 LISTENING。

## 步骤 5：构建荒岛世界

```bash
cd AISmallWorld                  # 回到项目根目录
python scripts/build_world.py    # 通过 telnet 连接 Evennia，构建所有房间和出口
```

必须看到 `BEACH:` 输出中包含 `#10`。看到 `NOT FOUND` 说明构建失败。

> 绝对不能用 `dh_client` 执行 `@py` 命令来构建世界。`dh_client` 通过 WebSocket 连接，`@py` 内部调用 `build()` 会触发 Twisted reactor 冲突导致无响应。必须通过 telnet（端口 4000）执行。

## 步骤 6：创建 AI 玩家账号和角色

```bash
cd AISmallWorld                       # 回到项目根目录
python scripts/create_player.py       # 创建 yueying 账号和游戏角色，放置到白色沙滩
```

必须看到 `CREATE OK` 和 `CHAR OK`。

## 步骤 7：配置 CLI

```bash
cd AISmallWorld    # 回到项目根目录（配置文件必须放在这里，其他位置找不到）
cat > dh_config.json << 'EOF'
{
    "evennia": {
        "host": "localhost",
        "port": 4002,
        "account": "yueying",
        "password": "Yueying#2026Island"
    }
}
EOF
```

## 步骤 8：验证

```bash
cd AISmallWorld                                       # 项目根目录
PYTHONPATH=src python -m dh_client.dh look             # src 加入模块搜索路径，执行 look 指令
```

看到「白色沙滩」三个字 = 部署成功。

> `PYTHONPATH=src` 不能省。`dh_client` 模块在 `src/` 目录下，不设会报 `ModuleNotFoundError`。

## 步骤 9：人类开发者访问

浏览器打开：

```
http://localhost:4001
```

用 `mofi / Mofi#2026Island` 登录即可。

## CLI 用法

所有命令在项目根目录执行，必须带 `PYTHONPATH=src`：

```bash
cd AISmallWorld
PYTHONPATH=src python -m dh_client.dh look              # 看周围
PYTHONPATH=src python -m dh_client.dh status            # 查看状态
PYTHONPATH=src python -m dh_client.dh inventory         # 查看背包
PYTHONPATH=src python -m dh_client.dh help              # 查看所有指令
PYTHONPATH=src python -m dh_client.dh "move n"          # 往北走
PYTHONPATH=src python -m dh_client.dh search            # 搜索
PYTHONPATH=src python -m dh_client.dh "get 石头"        # 捡起物品
PYTHONPATH=src python -m dh_client.dh research          # 研究配方
PYTHONPATH=src python -m dh_client.dh recipes           # 查看已学配方
PYTHONPATH=src python -m dh_client.dh "make 石刃"       # 制作物品
PYTHONPATH=src python -m dh_client.dh drink             # 喝水
PYTHONPATH=src python -m dh_client.dh eat "椰肉"        # 吃东西
PYTHONPATH=src python -m dh_client.dh rest              # 休息
PYTHONPATH=src python -m dh_client.dh "listen 大海"     # 听
PYTHONPATH=src python -m dh_client.dh "smell 溪流"      # 闻
PYTHONPATH=src python -m dh_client.dh "touch 沙子"      # 触摸
```

## 关闭服务

```bash
cd AISmallWorld/evennia_game   # 进入 Evennia 实例目录
evennia stop                    # 停止服务
```

## 完全重来

如果部署过程中出问题，直接从**步骤 3** 重新开始。步骤 3 会自动停服务、清数据库、重新初始化。

## 端口说明

| 端口 | 用途 |
|------|------|
| 4000 | telnet（自动化脚本用） |
| 4001 | Web 浏览器（人类访问） |
| 4002 | WebSocket（dh_client CLI 用） |
