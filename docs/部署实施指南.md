等待出现 `Evennia running.` 提示。

```json

        "port": 4002,

        "password": "Wq#2026Island"

}
```

## 使用方式

```bash

# 基础操作
python -m dh_client.dh look              # 看周围
python -m dh_client.dh status            # 查看状态
python -m dh_client.dh inventory         # 查看背包
python -m dh_client.dh help              # 查看所有指令

# 移动
python -m dh_client.dh "move n"          # 往北走

# 采集
python -m dh_client.dh search            # 搜索
python -m dh_client.dh "get 石头"        # 捡起物品

# 制作
python -m dh_client.dh research          # 研究配方
python -m dh_client.dh recipes           # 查看已学配方

# 生存

python -m dh_client.dh "listen 大海"     # 听
python -m dh_client.dh "smell 溪流"      # 闻
python -m dh_client.dh "touch 沙子"      # 触摸
```

## 架构说明

```
Agent（Bash 调用）
    │
    │  python -m dh_client.dh "look"
    ▼
┌─ connector.py ──────────────────────┐
│  1. WebSocket 连接 Evennia:4002     │
│  2. client_options（纯文本模式）    │
│  3. 登录（等待 logged_in 信号）     │
│  4. 排空登录自动产生的消息          │
│  5. 发送命令                        │
│  6. 等待响应（消息间隔 ≥ 1秒）     │
│  7. 清洗输出（去 HTML）            │
│  8. 断开连接                        │
└─────────────────────────────────────┘
    │
    ▼
Evennia（端口 4002）
```

**每次调用独立**：连接 → 登录 → 发命令 → 收响应 → 断开。无状态，无后台进程。

## 故障排查

| 问题 | 原因 | 解决 |
|------|------|------|
| "无法连接到世界：Evennia 未启动或端口不对" | Evennia 未运行 | `evennia start` |
| "登录失败：连接超时" | 账号密码错误或账号未创建 | 检查 dh_config.json，确认账号已创建 |
| "命令执行超时" | Evennia 响应慢或命令卡住 | 重试即可，每次调用独立 |
| 输出乱码 | Windows 终端编码 | dh.py 已处理 UTF-8，如仍有问题检查终端编码 |
| "配置文件不存在" | dh_config.json 不在项目根目录 | 确认文件在 `AISmallWorld_v2/dh_config.json` |
