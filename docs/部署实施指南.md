# 部署实施指南：数字家园试炼场

**按顺序执行以下步骤，每一步都必须完成后再进入下一步。所有命令直接复制粘贴执行即可。**

## 步骤 1：克隆项目

```bash
git clone https://github.com/caoyc/AISmallWorld.git
cd AISmallWorld
```

## 步骤 2：创建 conda 环境并安装依赖

```bash
conda create -n aismallworld python=3.12 -y
conda activate aismallworld
pip install evennia websockets beautifulsoup4
```

> **坑①**：系统 Python 可能是 3.10，不满足 Evennia 要求的 3.12+。必须用 conda 创建 3.12 环境。
>
> **坑②**：`conda activate` 在某些 bash 环境下不可用。如果报错，用全路径替代：
> ```bash
> # 找到 conda 环境路径
> D:/ProgramData/CondaVenv/aismallworld/python.exe --version
> D:/ProgramData/CondaVenv/aismallworld/Scripts/evennia.exe --version
> ```
> 下文所有 `python` 替换为 `D:/ProgramData/CondaVenv/aismallworld/python.exe`，`evennia` 替换为 `D:/ProgramData/CondaVenv/aismallworld/Scripts/evennia.exe`。
>
> **坑③**：不要用 `conda run -n aismallworld ...`。`conda run` 不支持多行参数，输出中文时因 Windows GBK 编码会崩溃。

## 步骤 3：初始化数据库

```bash
cd evennia_game
mkdir -p server/logs
rm -f server/evennia.db3
evennia migrate
```

> **坑④**：`server/logs/` 目录在 `.gitignore` 中，clone 后不存在。不创建就启动会报"日志目录找不到"。
>
> **坑⑤**：如果之前启动过留下了损坏的数据库（比如 Account #1 被误删），必须 `rm -f server/evennia.db3` 删掉重来，否则 `evennia start` 会无限递归报错直到栈溢出。

## 步骤 4：启动 Evennia 并创建管理员

```bash
export EVENNIA_SUPERUSER_USERNAME=mofi
export EVENNIA_SUPERUSER_PASSWORD='Mofi#2026Island'
export EVENNIA_SUPERUSER_EMAIL=
evennia start
```

等待输出 `Server started.` 和 `Portal started.`。

确认三个端口就绪：

```bash
netstat -an | grep -E ":400[0-2].*LISTENING"
```

必须看到 4000、4001、4002 三个端口都在 LISTENING。

> **坑⑥**：`evennia start` 首次启动会检测 Account #1 是否存在。不存在则读取 `EVENNIA_SUPERUSER_*` 环境变量自动创建。如果不在 TTY 环境且没设这些变量，会无限跳过创建然后无限重试直到栈溢出。
>
> **坑⑦**：密码不能和用户名相似，不能是常见密码（如 `admin123`、`password`），否则 Django 密码校验会拒绝创建。上面预设的密码已经过验证。

## 步骤 5：构建荒岛世界

```bash
python << 'PYEOF'
import telnetlib, time, sys
sys.stdout.reconfigure(encoding='utf-8')

tn = telnetlib.Telnet('127.0.0.1', 4000, timeout=10)
time.sleep(1)
tn.read_very_eager()

tn.write('connect mofi Mofi#2026Island\n'.encode('utf-8'))
time.sleep(2)
tn.read_very_eager()

tn.write('@py from survival.v2.world.build_island import build; build()\n'.encode('utf-8'))
time.sleep(10)
resp = tn.read_very_eager().decode('utf-8', errors='replace')
print('BUILD:', resp[:500])

tn.write("@py from evennia.objects.models import ObjectDB; r=ObjectDB.objects.filter(db_key='白色沙滩'); print(r.first().dbref if r.exists() else 'NOT FOUND')\n".encode('utf-8'))
time.sleep(3)
resp = tn.read_very_eager().decode('utf-8', errors='replace')
print('BEACH:', resp)

tn.write('quit\n'.encode('utf-8'))
time.sleep(1)
tn.close()
PYEOF
```

必须看到 `BEACH:` 输出中包含 `#10`。看到 `NOT FOUND` 说明构建失败。

> **坑⑧**：绝对不能用 `dh_client` 执行 `@py` 命令来构建世界。`dh_client` 通过 WebSocket 连接，`@py` 内部调用 `build()` 会触发 Twisted reactor 冲突，导致无响应。必须通过 telnet（端口 4000）或 Web 界面（http://localhost:4001）执行。
>
> **坑⑨**：如果这一步没做，后面 `dh_client.dh look` 会看到 Evennia 默认的「Limbo」房间，而不是白色沙滩。看到 Limbo = 世界没构建。

## 步骤 6：创建 AI 玩家账号

```bash
python << 'PYEOF'
import telnetlib, time, sys
sys.stdout.reconfigure(encoding='utf-8')

tn = telnetlib.Telnet('127.0.0.1', 4000, timeout=10)
time.sleep(1)
tn.read_very_eager()

tn.write('create yueying Yueying#2026Island\n'.encode('utf-8'))
time.sleep(3)
resp = tn.read_very_eager().decode('utf-8', errors='replace')
print('CREATE:', resp)

tn.write('Y\n'.encode('utf-8'))
time.sleep(2)
resp = tn.read_very_eager().decode('utf-8', errors='replace')
print('CONFIRM:', resp)

tn.write('quit\n'.encode('utf-8'))
time.sleep(1)
tn.close()
PYEOF
```

必须看到 `CONFIRM:` 输出中包含 `A new account 'yueying' was created.`。

角色自动出生在白色沙滩（#10），无需手动传送。

## 步骤 7：配置 CLI

```bash
cd AISmallWorld
cat > dh_config.json << 'EOF'
{
    "evennia": {
        "host": "localhost",
        "port": 4002,
        "account": "yueying",
        "password": "Yueying#2026Island"
    }
}
EOF
```

> **坑⑩**：配置文件必须在项目根目录（`AISmallWorld/dh_config.json`），不是 `src/` 下，不是 `evennia_game/` 下。

## 步骤 8：验证

```bash
PYTHONPATH=src python -m dh_client.dh look
```

看到「白色沙滩」三个字 = 部署成功。

> **坑⑪**：`PYTHONPATH=src` 不能省。`dh_client` 模块在 `src/` 目录下，不设会报 `ModuleNotFoundError: No module named 'dh_client'`。
>
> **坑⑫**：如果看到「登录失败：连接超时」，说明 dh_config.json 中的账号密码与步骤 6 创建的不一致，或者 Evennia 没在运行。

## 步骤 9：人类开发者访问

浏览器打开：

```
http://localhost:4001
```

用 `mofi / Mofi#2026Island` 或 `yueying / Yueying#2026Island` 登录即可。

## CLI 用法

所有命令在项目根目录执行，必须带 `PYTHONPATH=src`：

```bash
cd AISmallWorld

PYTHONPATH=src python -m dh_client.dh look              # 看周围
PYTHONPATH=src python -m dh_client.dh status            # 查看状态
PYTHONPATH=src python -m dh_client.dh inventory         # 查看背包
PYTHONPATH=src python -m dh_client.dh help              # 查看所有指令
PYTHONPATH=src python -m dh_client.dh "move n"          # 往北走
PYTHONPATH=src python -m dh_client.dh search            # 搜索
PYTHONPATH=src python -m dh_client.dh "get 石头"        # 捡起物品
PYTHONPATH=src python -m dh_client.dh research          # 研究配方
PYTHONPATH=src python -m dh_client.dh recipes           # 查看已学配方
PYTHONPATH=src python -m dh_client.dh "make 石刃"       # 制作物品
PYTHONPATH=src python -m dh_client.dh drink             # 喝水
PYTHONPATH=src python -m dh_client.dh eat "椰肉"        # 吃东西
PYTHONPATH=src python -m dh_client.dh rest              # 休息
PYTHONPATH=src python -m dh_client.dh "listen 大海"     # 听
PYTHONPATH=src python -m dh_client.dh "smell 溪流"      # 闻
PYTHONPATH=src python -m dh_client.dh "touch 沙子"      # 触摸
```

## 关闭服务

```bash
cd AISmallWorld/evennia_game
evennia stop
```

## 完全重来

如果部署过程中出问题，从零开始：

```bash
cd AISmallWorld/evennia_game
evennia stop
rm -f server/evennia.db3
```

然后从**步骤 3** 重新开始。
