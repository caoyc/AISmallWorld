# 演讲和脚本模式开发文档

## 概述

本文档目录包含演讲模式和脚本模式的设计方案和实现文档。

## 模式说明

### 演讲模式
- **状态**：待实现
- **描述**：手动选择说话人，直接输入对话内容，不调用大模型
- **文档**：[演讲模式设计方案](./演讲模式设计方案.md)

### 脚本模式
- **状态**：规划中（后续实现）
- **描述**：基于WebGAL的脚本播放模式，支持预设对话流程
- **文档**：[脚本模式设计方案](./脚本模式设计方案.md)

## 开发优先级

1. **第一阶段**：演讲模式
   - 基础功能实现
   - 说话人选择
   - 消息发送
   - 语音播放

2. **第二阶段**：脚本模式（后续实现）
   - 脚本解析器
   - 播放引擎
   - UI实现
