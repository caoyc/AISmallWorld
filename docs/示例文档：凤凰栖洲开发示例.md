# 凤凰栖洲 - 开发示例指令文档

本文档提供完整的、可连续执行的开发指令流程，用于快速上手开发区开发，同时完成凤凰栖洲的基础建设。

**主题**：热带河畔凤凰木传说浪漫场景

**格式说明**：
- 每行指令使用"!"前缀，对应标准指令语法（使用 `!` 前缀）
- 所有指令按顺序执行，无需手动判断
- 包含完整的进入/退出操作
- 覆盖主要指令类型（create, look, list, update, visit, return）

**设计原则**：
- 操作连续性：每个步骤都有明确的进入和退出
- 开发完整性：覆盖区域、场所、房间、物品、出口的创建
- 层级清晰：按照 World → Region → Place → Room 的顺序开发
- 示例完整：提供完整的开发流程作为参考
- 主题契合：围绕凤凰木传说、河畔、洲渚、缘分等浪漫元素

## 执行说明

### 执行方式
1. **逐条复制执行**：从阶段一开始，逐条复制指令到游戏客户端执行
2. **批量执行**：可以一次性复制整个阶段的所有指令，逐条执行
3. **使用 ! 前缀**：所有标准指令前添加 `!` 前缀，直接执行游戏指令
