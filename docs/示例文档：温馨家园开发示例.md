# 温馨家园 - 开发示例指令文档

本文档提供完整的、可连续执行的开发指令流程，用于快速上手开发区开发，同时完成温馨家园的基础建设。

**格式说明**：
- 每行指令使用"！"前缀，对应标准指令语法（使用 `!` 前缀）
- 所有指令按顺序执行，无需手动判断
- 包含完整的进入/退出操作
- 覆盖主要指令类型（create, look, list, update, visit, return）

**设计原则**：
- 操作连续性：每个步骤都有明确的进入和退出
- 开发完整性：覆盖区域、场所、房间、物品、出口的创建
- 层级清晰：按照 World → Region → Place → Room 的顺序开发
- 示例完整：提供完整的开发流程作为参考

## 后续开发建议

如果需要进一步开发温馨家园，可以考虑：

1. **NPC管理**：在房间中创建NPC（如管家、邻居等）
   ```
   !npc create name=管家 desc="负责管理家务的管家，和蔼可亲"
   ```

2. **角色管理**：创建和管理角色
   ```
   !char create name=小明 desc="一个活泼可爱的小男孩"
   ```

3. **图像更新**：为各个场所和房间更新图像
   ```
   !update image="温馨的客厅场景"
   ```

4. **物品交互**：使用物品（get、drop、wear等操作）
   ```
   !item get name=沙发
   !item drop name=沙发
   ```

5. **房间移动**：通过出口在不同房间之间移动（go指令）
   ```
   !go name=卧室门
   ```

6. **更多房间**：可以添加更多房间，如卫生间、储物间等
   ```
   !room create name=卫生间 desc="住宅区的卫生设施，干净整洁"
   ```

这些操作可以在Room层完成，使用相应的!指令即可。

## 注意事项

1. **标准指令语法**：本示例使用 `!` 前缀的标准指令语法，与 `$` 自然语言语法不同
2. **参数格式**：标准指令使用 `name=值`、`desc="值"` 等参数格式
3. **层级切换**：使用 `visit` 进入下一层，使用 `return` 返回上一层
4. **错误处理**：如果某条指令执行失败，查看控制台日志，修复后继续执行
5. **数据持久化**：所有创建的对象会保存到数据库，重启游戏后仍然存在
6. **指令对比**：
   - `$` 语法：`$创建一个名为客厅的房间`（自然语言）
   - `!` 语法：`!room create name=客厅 desc="..."`（标准指令）

## 快速参考

### 常用指令类型
- **创建区域**：`!create name=区域名 desc="描述"`
- **创建场所**：`!create name=场所名 desc="描述"`
- **创建房间**：`!room create name=房间名 desc="描述"`
- **创建物品**：`!item create name=物品名 desc="描述"`
- **创建出口**：`!exit create name=出口名 to=目标房间 desc="描述"`
- **查看**：`!look`
- **列表**：`!list`、`!room list`、`!item list`、`!exit list`
- **更新**：`!update desc="新描述"`
- **进入**：`!visit name=名称`
- **返回**：`!return`
- **移动**：`!go name=出口名`

### 层级关系
- **World层（荒野层）**：创建和管理区域
- **Region层（区域层）**：创建和管理场所
- **Place层（场所层）**：创建和管理房间
- **Room层（房间层）**：创建和管理物品、出口、NPC等

### 开发流程
1. 在World层创建区域
2. 进入Region层，创建场所
3. 进入Place层，创建房间
4. 进入Room层，创建物品和出口
5. 使用return逐层返回

### 指令语法对比

| 功能 | $ 语法（自然语言） | ! 语法（标准指令） |
|------|-------------------|-------------------|
| 创建区域 | `$创建一个名为温馨家园的区域` | `!create name=温馨家园 desc="..."` |
| 创建房间 | `$在当前场所建一个名为客厅的房间` | `!room create name=客厅 desc="..."` |
| 创建物品 | `$在客厅创建一个名为沙发的物品` | `!item create name=沙发 desc="..."` |
| 创建出口 | `$在客厅创建一个名为卧室门的出口，通往卧室` | `!exit create name=卧室门 to=卧室 desc="..."` |
| 查看信息 | `$查看当前区域信息` | `!look` |
| 列出对象 | `$列出当前区域的所有场所` | `!list` |
| 进入 | `$进入住宅区场所` | `!visit name=住宅区` |
| 返回 | `$返回区域层` | `!return` |
| 移动 | `$通过卧室门进入卧室` | `!go name=卧室门` |

---

**提示**：本示例文档使用标准指令语法（!前缀），适合熟悉游戏指令的用户。如果需要使用自然语言，可以参考"丘陵茶园开发示例"文档（使用$前缀）。
