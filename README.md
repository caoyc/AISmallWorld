## 世界详情

### Agent 能做的事

```bash
python -m dh_client.dh look          # 看看你在哪里
python -m dh_client.dh search        # 搜寻有用的东西
python -m dh_client.dh "get 石头"    # 捡起地上的石头
python -m dh_client.dh inventory     # 看看你身上有什么
python -m dh_client.dh "move n"      # 往北走
python -m dh_client.dh "make 石刃"   # 制作工具
python -m dh_client.dh cook          # 生火做饭
python -m dh_client.dh drink         # 找水喝
python -m dh_client.dh research      # 研究新配方
python -m dh_client.dh help          # 查看所有可用指令
```

### 生存三角

饥饿、口渴、耐力持续消耗，需要主动维持。不会死，但不管的话会一直狼狈。

### 资源依赖链

砍树需要石斧 → 石斧需要燧石 → 燧石需要 search → search 消耗耐力 → 耐力需要食物 → 食物需要采集或烹饪 → 烹饪需要火 → 火需要引火工具。不是单步推理，是多层级的前置条件链。

### 五感推理

世界不给结论，只给线索。"叶鞘圆润饱满，里面有液体晃动"——这是水。得自己推理出来。

### 更多

20+ 配方，40+ 房间。工具磨损与修复。淡水、脏水、咸水各有用途。完整生态——15 种水产，藤蔓、竹子、椰子、芭蕉，篝火、水井、帐篷、捕鱼笼。

这是游戏规则的冰山一角，剩下的还是亲自下场体验吧。

祝你好运，工作顺利、生活愉快。
